About

Slide through your content/images with ease using CSS transitions. No need to write code. In browsers that don't support CSS transitions, jQuery's animate is used.

Keyboard navigation is available when the element has focus. Simply use the left/right arrows keys.

The content slider will automatically respond to changes to the available screen width by re-flowing the active item into view. This behavior can be disabled by setting the reflow option to false.

This plugin uses the amazing imagesLoaded plugin when available to load all images before initializing. The flow-init class will be applied to the element while the images are being loaded. The init event will fire after the images are loaded. This makes it extremely easy to add a loading graphic, or perform tasks after the images are loaded.

Usage

Setup

Before getting started, make sure everything is setup properly. Below is a basic HTML template to use as an example for including the plugin into your projects.

<!DOCTYPE html>
<html>
<head>
    <title>Flow.js</title>
    <meta charset="utf-8">
    <link rel="stylesheet" href="./path/to/flow.css">
</head>
<body>
    <!-- Your HTML goes here -->
    <script src="./path/to/jquery.js"></script>
    <script src="./path/to/flow.js"></script>
</body>
</html>

Attention! This is just an example and contains the minimum requirements for including Flow.js into your project. All ./path/to references need to be replaced with proper paths.

Via data attributes

To initialize a plugin instance, simply add data-flow="init" to your element. Be sure to add the flow class for proper styling.

<div data-flow="init" class="flow">
    ...
</div>

Via Javascript

$('.flow').flow();

Accessing instances

Plugin instances are stored in the flowjs key on each element's data store and can easily be accessed by using jQuery's data method.

var myFlow = $('.flow').data('flowjs');

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-start="3".

Name Type Default Description
start number 1

The index of the first item to display.

Notice! The start index is one-based. A value of 1 corresponds to the first item. Negative, zero and out-of-range positions will be ignored.

speed number 500 The amount of time (in milliseconds) that corresponds to the CSS transition speed specified in flow.css. This value is used to ensure transitions are completed and as the animation speed for browsers that don't support CSS transitions.
reflow boolean true Automatically re-flow the active item into view when the screen size changes.

Methods

All methods can be executed with the data-flow attribute and require a data-target or href attribute whose value must correspond to the ID of a Flow.js element. See below for examples.

.flow(options)

Initializes an instance with an optional options object.

$('.flow').flow({
    start: 3
});
.flow('next')

Flow to the next item.

Via data attribute

<a data-flow="next" href="#myFlow">Next</a>
.flow('prev')

Flow to the previous item.

Via data attribute

<a data-flow="prev" data-target="#myFlow">Previous</a>
.flow('to', position)

Flow to the item at the specified position.

Be aware! The position argument is one-based. A value of 1 corresponds to the first item. Negative, zero and out-of-range positions will be ignored.

Via data attribute

<a data-flow="to" data-args="3" href="#myFlow">Go to item 3</a>

The data-args attribute is used to pass arguments to a method.

Events

Name Description
init Fires after an instance has been initialized.
flow

Fires before a new item is flowed into view.

Property Description
relatedTarget The item element being flowed into view.
relatedIndex The index of the item element being flowed into view.

Use e.preventDefault() on this event to cancel the cycling of an item.
flowed

Fired after a new item has been flowed into view.

Property Description
relatedTarget The item element that was flowed into view.
relatedIndex The index of the item element flowed into view.

Basic

The following example will start with the first item. Click on the arrows, or focus the element and use keyboard arrows, to move to the next item.

HTML
<div id="basic-flow" data-flow="init" class="flow">
    <div class="flow-inner">
        <div class="item">
            <img src="http://lorempixel.com/500/300/sports" alt="">
        </div>
        <div class="item">
            <img src="http://lorempixel.com/500/300/animals" alt="">
        </div>
        <div class="item">
            <img src="http://lorempixel.com/500/300/food" alt="">
        </div>
    </div>
</div>
